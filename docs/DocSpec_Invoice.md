## 2. Состояния

| Код | Отображаемое название | Описание |
| --- | --- | --- |
| `draft` | Черновик | Счёт создан, редактирование разрешено |
| `sent` | Выставлен | Счёт отправлен клиенту |
| `paid` | Оплачен | Оплата зафиксирована |
| `overdue` | Просрочен | Срок оплаты истёк |
| `cancelled` | Аннулирован | Счёт аннулирован |

## 3. Переходы между состояниями

| Из → В | Условие | Кто выполняет | Проверки |
| --- | --- | --- | --- |
| `draft` → `sent` | Счёт выставлен клиенту | admin, svc_mgr, accountant | Клиент указан; хотя бы одна позиция |
| `sent` → `paid` | Оплата получена | admin, accountant | — |
| `sent` → `overdue` | Авто: `due_date < now()` и не оплачен | Система (cron) | due_date не NULL |
| `overdue` → `paid` | Оплата получена с опозданием | admin, accountant | — |
| `draft` → `cancelled` | Аннулирование черновика | admin, accountant | — |
| `sent` → `cancelled` | Аннулирование выставленного | admin, accountant | — |

## 7. Проверки при переходах

| Переход | Проверки |
| --- | --- |
| `draft` → `sent` | `client_id` ≠ NULL; `invoice_items` не пустой; `total_amount ≥ 0` |
| `sent` → `paid` | — |
| `sent` → `overdue` | `due_date` не NULL; `due_date < now()`; статус = `sent` |
| Сохранение позиций | `quantity > 0`; `unit_price ≥ 0`; если `item_type=service` то `service_id` обязателен; если `item_type=part` то `part_id` обязателен и `spare_parts.unit_price > 0` (BR-F-122); `item_type=manual` — `service_id` и `part_id` не заполняются |
| Создание из акта | Акт содержит позиции (BR-F-118); если заявка гарантийная — предупреждение (BR-P-005) |

## 8. Правило расчёта НДС

**НДС считается «в т.ч.» (включён в цену) — НЕ добавляется к итогу счёта.**

| Поле | Формула | Пример (сумма позиций 1 220 [системная валюта], ставка 22%) |
| --- | --- | --- |
| `total_amount` | Σ позиций | 1 220,00 [системная валюта] |
| `vat_amount` | `total / (100 + rate) × rate` | 1 220 / 122 × 22 = **220,00 [системная валюта]** |
| `subtotal` | `total_amount − vat_amount` | 1 220 − 220 = **1 000,00 [системная валюта]** |

> Цены на услуги и запчасти вводятся **с НДС**. `vat_amount` и `subtotal` отображаются справочно на печатной форме счёта.
