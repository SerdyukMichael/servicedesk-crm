# UC-1005: Поиск оборудования

**Версия:** 1.0 | **Дата:** 26.03.2026 | **Статус:** На проверке

---

## Паспорт

| Поле | Значение |
| --- | --- |
| **ID** | UC-1005 |
| **Модуль** | Модуль 10: Учёт оборудования |

---

## Основной поток

1. Пользователь открывает раздел **«Оборудование»**.
2. Система отображает таблицу всего оборудования (активного по умолчанию).
3. Пользователь вводит поисковый запрос и/или применяет фильтры.
4. Система отображает отфильтрованный список.
5. Пользователь кликает на строку — открывается паспорт оборудования.

---

## Доступные параметры поиска / фильтрации (BR-F-1006)

| Параметр | Тип | Примечание |
| --- | --- | --- |
| Серийный номер | Текст | Точное и частичное совпадение |
| Модель | Выбор | Из справочника |
| Клиент | Поиск + выбор | — |
| Адрес установки | Текст | Частичное совпадение |
| Гарантийный статус | Мультивыбор | На гарантии / Истекает / Истекла |
| Статус оборудования | Мультивыбор | Активно / На ремонте / Списано / Передано |

**Карточка клиента** также отображает список его оборудования с колонками (BR-F-1005):
модель, серийный номер, адрес установки, гарантийный статус, дата последнего ТО, количество ремонтов.

---

## Ключевые данные (параметры поиска)

| Параметр | Тип | Описание |
| --- | --- | --- |
| serial_number | varchar | Частичное совпадение (LIKE) |
| model_id | FK → equipment_models | Точный выбор из справочника |
| client_id | FK → clients | Точный выбор |
| address | varchar | Частичное совпадение |
| warranty_status | enum | `active`, `expiring`, `expired` |
| equipment_status | enum | `active`, `in_repair`, `written_off`, `transferred` |

---

## Исключения

**ИП-1: Поиск не дал результатов**
→ Система отображает: *«Оборудование не найдено. Измените параметры поиска»*. Кнопка «Сбросить фильтры» доступна.

**ИП-2: Нет прав на просмотр оборудования**
→ Раздел «Оборудование» недоступен в меню. Прямой URL возвращает 403.

## Acceptance Criteria

```gherkin
Feature: Поиск оборудования

  Scenario: Поиск по частичному серийному номеру
    When пользователь вводит «SN-12»
    Then отображаются все записи, серийный номер которых содержит «SN-12»

  Scenario: Фильтрация по гарантийному статусу
    When пользователь выбирает «Гарантия истекает»
    Then отображается только оборудование с warranty_status = expiring

  Scenario: Список оборудования клиента
    Given клиент «Банк Альфа» имеет 5 единиц оборудования
    When пользователь открывает карточку «Банк Альфа»
    Then в разделе «Оборудование» отображается таблица из 5 строк
    And по каждой строке: модель, SN, адрес, гарантийный статус, дата ТО, количество ремонтов
```

---

## Примечания

- Поиск работает по всему оборудованию системы (не только активному). Списанное и переданное включается при выборе соответствующего фильтра статуса.
