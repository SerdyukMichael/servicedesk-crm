## Основной поток

1. Пользователь нажимает на **иконку колокольчика** (с бейджем счётчика непрочитанных).
2. Система отображает выпадающую ленту: последние 20 уведомлений, хронологически (новые сверху).
3. Непрочитанные уведомления выделены визуально (жирный шрифт / фоновый цвет).
4. Пользователь кликает на уведомление:
   - Уведомление помечается «Прочитано»
   - Система переходит к связанному объекту (заявке, оборудованию и т.д.)
5. Пользователь нажимает **«Отметить все прочитанными»** — все уведомления в ленте помечаются прочитанными.
6. Пользователь нажимает **«Все уведомления»** — открывается полная страница с историей и фильтрами.

## Состав уведомления

| Поле | Описание |
|---|---|
| Иконка / тип | Визуальный маркер типа события |
| Заголовок | Краткое описание события |
| Время | Относительное («5 минут назад») и абсолютное при наведении |
| Ссылка | Переход к связанному объекту |
| Статус | Прочитано / Непрочитано |

## Полная страница истории уведомлений (BR-F-1403)

Доступные фильтры:
- По типу события (мультивыбор)
- По статусу (Все / Непрочитанные / Прочитанные)
- По дате (диапазон)

## Acceptance Criteria

```gherkin
Feature: Лента in-app уведомлений

  Scenario: Получение уведомления о назначении заявки
    Given инженеру «Петрову» назначена заявка #42
    When «Петров» смотрит на иконку колокольчика
    Then бейдж показывает 1 непрочитанное
    And в ленте есть уведомление «Вам назначена заявка #42»

  Scenario: Переход к объекту при клике
    Given в ленте есть уведомление о заявке #42
    When пользователь кликает на уведомление
    Then открывается карточка заявки #42
    And уведомление помечено «Прочитано»

  Scenario: Доставка push за 1 минуту (BR-F-1402)
    Given инженеру назначена заявка
    Then push-уведомление доставлено в мобильное приложение в течение 1 минуты
```

---

## Примечания / Бизнес-правила

- **BR-F-1402:** push — ≤ 1 минуты после события; email — ≤ 5 минут.
- **BR-F-1403:** история хранится без ограничения по времени; пользователь может фильтровать и отмечать прочитанными.
- In-app уведомления нельзя отключить (BR-F-1401).
