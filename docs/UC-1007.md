# UC-1007: Оформить возврат оборудования от клиента

**Версия:** 1.0 | **Дата:** 26.03.2026 | **Статус:** На проверке

---

## Паспорт

| Поле | Значение |
|---|---|
| **ID** | UC-1007 |
| **Модуль** | Модуль 10: Учёт оборудования |
| **Основной актор** | Руководитель сервиса |
| **Второстепенные акторы** | Система (инициирует рекламацию вендору при гарантийном возврате → BR-F-1111) |
| **Связанные BR-F** | BR-F-1010, BR-F-1111 |

| **Приоритет** | Should |
| **Предусловия** | Паспорт оборудования существует, статус — **«Активно»**; пользователь авторизован |
| **Постусловия** | Оборудование отвязано от клиента; статус изменён; история передачи зафиксирована |
| **Триггер** | Руководитель сервиса открывает паспорт оборудования → **«Оформить возврат»** |

---

## Типы возврата

| Тип | Описание | Дальнейшие действия |
|---|---|---|
| Гарантийная замена | Дефект в гарантийный период | Система предлагает создать рекламацию вендору (→ BR-F-1111) |
| Выкуп б/у | Клиент продаёт оборудование обратно | — |

---

## Основной поток

1. Руководитель сервиса открывает паспорт → нажимает **«Оформить возврат»**.
2. Выбирает тип возврата.
3. Указывает дату возврата и причину.
4. Подтверждает.
5. Система:
   - Отвязывает оборудование от клиента (сохраняет историю: дата, клиент, тип возврата)
   - Меняет статус на **«Активно»** (без клиента) или **«Списано»** (при выкупе б/у, если требуется)
6. Если тип = «Гарантийная замена»: система предлагает *«Создать рекламацию вендору?»*. При согласии → открывается форма рекламации (→ Модуль 11).

## Acceptance Criteria

```gherkin
Feature: Возврат оборудования

  Scenario: Гарантийный возврат с рекламацией
    Given оборудование SN-12345 «На гарантии» у клиента «Банк Альфа»
    When оформляется возврат типа «Гарантийная замена»
    Then оборудование отвязывается от «Банк Альфа»
    And система предлагает создать рекламацию вендору

  Scenario: История возвратов сохраняется
    Given возврат оформлен
    Then на вкладке «История передач» паспорта есть запись о возврате
```

---

## Примечания

- История всех передач и возвратов хранится на вкладке «История передач» паспорта. Записи не удаляются (BR-R-009).
