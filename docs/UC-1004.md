# UC-1004: Прикрепить документы к паспорту оборудования

**Версия:** 1.0 | **Дата:** 26.03.2026 | **Статус:** На проверке

---

## Паспорт

| Поле | Значение |
|---|---|
| **ID** | UC-1004 |
| **Модуль** | Модуль 10: Учёт оборудования |
| **Основной актор** | Руководитель сервиса |
| **Связанные BR-F** | BR-F-1004 |
| **Приоритет** | Must |
| **Предусловия** | Паспорт оборудования существует; пользователь авторизован |
| **Постусловия** | Документ сохранён и привязан к паспорту |
| **Триггер** | Пользователь открывает паспорт оборудования → вкладка **«Документы»** → «Прикрепить» |

---

## Основной поток

1. Пользователь открывает паспорт → вкладка **«Документы»**.
2. Система отображает список ранее прикреплённых документов.
3. Пользователь нажимает **«Прикрепить документ»**, выбирает тип документа и файл.
4. Система валидирует: формат (PDF, JPEG, PNG, DOCX, XLSX, ZIP), размер (≤ 20 МБ).
5. Система сохраняет файл и отображает в списке.

---

## Типы документов

| Тип | Описание |
|---|---|
| Акт ввода в эксплуатацию | Подписанный акт приёмки оборудования |
| Гарантийный талон | Документ гарантии |
| Договор обслуживания | Привязка договора к конкретной единице |
| Прочее | Любой другой документ |

---

## Исключения

**ИП-1: Недопустимый формат или превышен размер**
→ Аналогично UC-907: сообщение об ошибке, остальные файлы загружаются.

```gherkin
Feature: Документы паспорта оборудования

  Scenario: Прикрепление акта ввода в эксплуатацию
    Given паспорт оборудования SN-12345
    When пользователь прикрепляет «act_vvod.pdf» (тип = Акт ввода в эксплуатацию)
    Then файл отображается в списке документов с типом «Акт ввода в эксплуатацию»
```

---

## Примечания

- Правила прикрепления файлов единые для всей системы (BR-F-1004): PDF, JPEG, PNG, DOCX, XLSX, ZIP, ≤ 20 МБ.
- К одному типу документа можно прикрепить несколько файлов (например, несколько страниц гарантийного талона).
