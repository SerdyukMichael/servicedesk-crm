# UC-1004: Прикрепить документы к паспорту оборудования

**Версия:** 1.0 | **Дата:** 26.03.2026 | **Статус:** На проверке

---

## Паспорт

| Поле | Значение |
| --- | --- |
| **ID** | UC-1004 |
| **Модуль** | Модуль 10: Учёт оборудования |
| **Основной актор** | Руководитель сервиса |
| **Связанные BR-F** | BR-F-1004 |
| **Приоритет** | Must |
| **Предусловия** | Паспорт оборудования существует; пользователь авторизован |
| **Постусловия** | Документ сохранён и привязан к паспорту |
| **Триггер** | Пользователь открывает паспорт оборудования → вкладка **«Документы»** → «Прикрепить» |

---

## Основной поток

1. Пользователь открывает паспорт → вкладка **«Документы»**.
2. Система отображает список ранее прикреплённых документов.
3. Пользователь нажимает **«Прикрепить документ»**, выбирает тип документа и файл.
4. Система валидирует: формат (PDF, JPEG, PNG, DOCX, XLSX, ZIP), размер (≤ 20 МБ).
5. Система сохраняет файл и отображает в списке.

---

## Типы документов

| Тип | Описание |
| --- | --- |
| Акт ввода в эксплуатацию | Подписанный акт приёмки оборудования |
| Гарантийный талон | Документ гарантии |
| Договор обслуживания | Привязка договора к конкретной единице |
| Прочее | Любой другой документ |

---

## Альтернативные потоки

**АП-1: Прикрепление нескольких файлов одновременно**
Пользователь выбирает несколько файлов за один раз.
Система валидирует каждый независимо: файлы с ошибкой помечаются, остальные загружаются.

**АП-2: Удаление ранее прикреплённого документа**
Пользователь нажимает «Удалить» рядом с документом, подтверждает — документ удаляется физически (исключение из BR-R-009: документы оборудования допускают физическое удаление).
2. Система запрашивает подтверждение.
3. При подтверждении — документ удаляется физически (исключение из BR-R-009: документы оборудования допускают физическое удаление).

---

## Исключения

**ИП-1: Недопустимый формат**
→ *«Недопустимый формат. Разрешены: PDF, JPEG, PNG, DOCX, XLSX, ZIP»*. Остальные файлы загружаются.

**ИП-2: Превышен размер файла**
→ *«Файл [имя] превышает 20 МБ»*. Остальные файлы загружаются.

**ИП-3: Тип документа не выбран**
→ Кнопка «Прикрепить» неактивна до выбора типа из выпадающего списка.

## Acceptance Criteria

```gherkin
Feature: Документы паспорта оборудования

  Scenario: Прикрепление акта ввода в эксплуатацию
    Given паспорт оборудования SN-12345
    When пользователь прикрепляет «act_vvod.pdf» (тип = Акт ввода в эксплуатацию)
    Then файл отображается в списке документов с типом «Акт ввода в эксплуатацию»
    And доступен для скачивания

  Scenario: Несколько файлов одного типа
    Given паспорт SN-12345
    When пользователь прикрепляет 3 файла с типом «Гарантийный талон»
    Then все 3 файла отображаются в разделе «Гарантийный талон»

  Scenario: Отклонение недопустимого формата
    When пользователь выбирает файл «video.avi» (тип = Прочее)
    Then система показывает «Недопустимый формат»
    And файл не сохраняется

  Scenario: Тип документа обязателен
    When пользователь пытается прикрепить файл без выбора типа
    Then кнопка «Прикрепить» неактивна

  Scenario: Удаление документа
    Given к паспорту SN-12345 прикреплён «warranty.pdf»
    When пользователь нажимает «Удалить» и подтверждает
    Then файл исчезает из списка документов
```

---

## Примечания

- Правила прикрепления файлов единые для всей системы (BR-F-1004): PDF, JPEG, PNG, DOCX, XLSX, ZIP, ≤ 20 МБ.
- К одному типу документа можно прикрепить несколько файлов (например, несколько страниц гарантийного талона).
- Физическое удаление документов оборудования допустимо (в отличие от вложений заявок и комментариев).
