## Основной поток

1. Пользователь открывает вкладку **«Файлы»** в карточке заявки.
2. Система отображает список ранее прикреплённых файлов (имя, тип, размер, автор, дата).
3. Пользователь нажимает **«Прикрепить файл»** и выбирает файл(ы) из галереи / файловой системы устройства.
4. Система валидирует каждый файл: формат (PDF, JPEG, PNG, DOCX, XLSX, ZIP) и размер (≤ 20 МБ).
5. Система загружает файлы и отображает их в списке.

---

## Альтернативные потоки

**АП-1: Съёмка фото напрямую с камеры мобильного устройства**

Применяется только в мобильном приложении инженера.

1. Инженер нажимает **«Сделать фото»** (кнопка рядом с «Прикрепить файл»).
2. Приложение запрашивает разрешение на доступ к камере (при первом использовании).
3. Открывается камера устройства.
4. Инженер снимает фото и нажимает **«Использовать»** (или делает несколько снимков подряд).
5. Система автоматически получает фото в формате JPEG, проверяет размер (≤ 20 МБ).
6. Файл сохраняется и отображается в списке вложений с меткой источника `camera`.
7. Геолокация и timestamp съёмки сохраняются в EXIF-метаданных файла (если разрешено устройством).

---

## Исключения

**ИП-1: Недопустимый формат**
→ Файл с ошибкой подсвечивается: *«Недопустимый формат. Разрешены: PDF, JPEG, PNG, DOCX, XLSX, ZIP»*.

**ИП-2: Превышен размер файла**
→ *«Файл [имя] превышает 20 МБ»*. Остальные файлы загружаются.

**ИП-3: Камера недоступна или доступ запрещён**
→ Приложение показывает: *«Нет доступа к камере. Разрешите доступ в настройках устройства»*.
→ Кнопка «Сделать фото» остаётся доступной — при повторном нажатии снова запрашивает разрешение.

## Примечания / Бизнес-правила

- Правила прикрепления файлов единые для всей системы: PDF, JPEG, PNG, DOCX, XLSX, ZIP; ≤ 20 МБ на файл.
- Съёмка с камеры доступна только в мобильном приложении. В веб-интерфейсе кнопка «Сделать фото» не отображается — только «Прикрепить файл».
- Фото с камеры сохраняются с `source = camera`; фото из галереи — `source = gallery`. Оба типа доступны для скачивания.
- Файлы из акта выполненных работ (фото UC-904) хранятся отдельно от общих вложений заявки, но отображаются в той же вкладке «Файлы» с меткой «Из акта».
