# RACI-матрица — ServiceDesk CRM

**Версия:** 1.0 | **Дата:** 27.03.2026 | **Статус:** Актуальна

---

## Обозначения

| Символ | Роль | Описание |
|--------|------|----------|
| **R** | Responsible | Исполнитель — выполняет задачу |
| **A** | Accountable | Владелец — принимает решение, несёт ответственность за результат |
| **C** | Consulted | Консультант — даёт экспертную обратную связь до/в процессе |
| **I** | Informed | Информируется — получает уведомление о результате |
| —    | —           | Не участвует в задаче |

> На каждую задачу — ровно один **A**. Может быть несколько **R**, **C**, **I**.

## Примечания

- **R·A** в одной ячейке означает, что исполнитель одновременно является и владельцем задачи (типично для небольших команд).
- RACI не покрывает оперативные задачи (баг-фиксы, хотфиксы) — для них используется отдельный процесс.
- При масштабировании команды (несколько BE / FE) роль **A** для разработки остаётся за TL, а **R** распределяется по разработчикам.
- Матрица актуальна для фаз 2–6 согласно [CLAUDE.md](../CLAUDE.md#фазы-разработки).
