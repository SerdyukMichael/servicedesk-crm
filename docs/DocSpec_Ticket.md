## 2. Состояния

| Код | Отображаемое название | Описание |
| --- | --- | --- |
| `new` | Новая | Заявка зарегистрирована, инженер не назначен |
| `assigned` | Назначена | Инженер назначен, работы не начаты |
| `in_progress` | В работе | Инженер приступил к выполнению |
| `waiting_part` | Ожидание запчасти | Работы приостановлены, ждём запчасть |
| `completed` | Выполнена | Инженер завершил работы, акт готов |
| `on_review` | На проверке | Передана на проверку руководителю |
| `closed` | Закрыта | Работы приняты, заявка закрыта |
| `cancelled` | Отменена | Заявка отменена с указанием причины |

---

## 3. Переходы между состояниями

| Из → В | Условие | Кто выполняет | Проверки |
| --- | --- | --- | --- |
| `new` → `assigned` | Назначен инженер | admin, svc_mgr | assigned_to заполнен |
| `assigned` → `in_progress` | Инженер начал работы | admin, svc_mgr, engineer⁷ | — |
| `in_progress` → `waiting_part` | Нет нужной запчасти | admin, svc_mgr, engineer⁷ | — |
| `waiting_part` → `in_progress` | Запчасть получена | admin, svc_mgr, engineer⁷ | — |
| `in_progress` → `completed` | Работы завершены | admin, svc_mgr, engineer⁷ | Акт выполненных работ существует и не черновик |
| `completed` → `on_review` | Передача на проверку | admin, svc_mgr | — |
| `on_review` → `closed` | Проверка пройдена | admin, svc_mgr | — |
| `on_review` → `in_progress` | Возврат на доработку | admin, svc_mgr | — |
| `any` → `cancelled` | Отмена заявки | admin, svc_mgr | cancellation_reason не пустое; нельзя отменить closed |

> ⁷ Инженер управляет статусом только заявок, назначенных на него.

## 7. Проверки при переходах

| Переход | Проверки |
| --- | --- |
| `new` → `assigned` | `assigned_to` ≠ NULL; инженер активен (`is_active=true`); если инженер в отпуске — предупреждение (не блокирует) |
| `in_progress` → `completed` | Существует акт выполненных работ (`work_acts`) для этой заявки с `is_draft=false` (BR-R-004) |
| `on_review` → `closed` | — |
| `any` → `cancelled` | `cancellation_reason` не пустое (мин. 5 символов); текущий статус ≠ `closed` |
| Списание запчасти | Только при наличии привязанной заявки (BR-R-005) |
