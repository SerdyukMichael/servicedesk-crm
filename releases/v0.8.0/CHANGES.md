# v0.8.0 — Редактирование акта выполненных работ

**Дата:** 08.04.2026

---

## Frontend

- Кнопка **«Редактировать акт»** на странице заявки — появляется когда акт существует и не подписан
- Форма создания акта переиспользуется для редактирования: предзаполняется описанием и позициями существующего акта
- Кнопка «Создать акт» меняется на «Сохранить изменения» в режиме редактирования
- Исправлен тип `WorkAct`: убраны несуществующие поля `signed_by_engineer`/`signed_by_client`, добавлен `signed_by: number | null`
- Добавлен хук `useUpdateWorkAct`

## Backend

- Новая схема `WorkActUpdate` (описание, время, позиции — все опциональны)
- Эндпоинт `PATCH /api/v1/tickets/{ticket_id}/work-act` — обновляет поля акта; позиции заменяются целиком (delete + insert)
- Возвращает **403 Forbidden** если акт подписан (`signed_by IS NOT NULL`)
- Доступен ролям: `engineer`, `svc_mgr`, `admin`

## Database

Миграций нет — только изменения в коде.
