# ServiceDesk CRM

Система автоматизации сервисной компании по обслуживанию банкоматов и платёжного оборудования.

## Требования 
Приложение автоматизирует работы фирмы по продаже и обслуживанию дорогостоящего оборудования - банкоматы и карт-машины фирмы Matica.

Приложение должно: 
1. вести сделки по продаже оборудования клиентам
2. вести историю взаимодействия с клиентами
3. вести историю обслуживания оборудования у клиентов
4. вести учет оборудования у клиентов
5. выставлять счета клиентам за обслуживание оборудования
6. вести расчеты с клиентами и вендорами 

нефункциональные требования:
1. Приложение должно иметь браузерную и мобильную версию
2. приложение должно иметь базу данных mysql
3. сервер приложений должен работать на python 
4. приложение должно базироваться на арендованном сервере  
5. Приложение должно соответствовать стандартам безопасности для интернет-приложений


## Модули системы

- 👥 **Клиенты** — CRM, история взаимодействий
- 🏧 **Оборудование** — учёт у клиентов, паспорта устройств
- 🔧 **Заявки** — ремонты, ТО, назначение инженеров
- 📦 **Склад** — запчасти, приход/расход
- 💰 **Финансы** — счета, акты, документы продажи
- 🏭 **Вендоры** — закупки оборудования и запчастей
- 📊 **Отчёты** — дашборд, аналитика

## Технологии

| Слой | Технология |
|------|-----------|
| БД | MySQL 8.0 |
| Backend | Python 3.11 + FastAPI |
| Веб | React + TypeScript |
| Мобильное | React Native |
| Деплой | Docker + Nginx |

## Быстрый старт

```bash
git clone https://github.com/SerdyukMichael/servicedesk-crm.git
cd servicedesk-crm
docker-compose up -d mysql
cd backend && python -m venv venv && venv\Scripts\activate
pip install -r requirements.txt
cp .env.example .env
alembic upgrade head
uvicorn app.main:app --reload
```

API документация: http://localhost:8000/docs
